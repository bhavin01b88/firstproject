Python Program to Calculate the Sum of Natural Numbers

To computer the sum of natural numbers from 1 to n (entered by the user), loops can be used.You will learn how to use For loop and While loop to solve this problem.

The positive numbers 1,2,3...are known as natural numbers.The programs below takes a positive integer (let say n) as an input from the user and calculate the sum up to n

Example.Read a number and calculate the Sum of it.s digit

SOURCE CODE
	# Python program to find the sum of natural numbers up to n where n is provided by user
	num = 16

	#num = int(input("Enter a number: "))

	if num < 0:
		print("Enter a positive number")
	else:
		sum = 0
	# use while loop to iterate un till zero
	while(num > 0):
		sum += num
		num -= 1
		print("The sum is",sum)

Output

	The Sum is 136

The above program takes the input from the user and store in variable n.Then, While loop is used to calculate the sum upto the given number.
