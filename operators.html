OPERATORS

An operator is a symbol that tells the computer to perform certain mathematical or logical manipulation.It can operate on integer,character and real numbers.Python operators can be classified into various categories based on their utility and action.They include-

1.Arithmetic Operators
2.Relational operators
3.Logical Operators
4.Assignment Operators
5.Increment or Decrement operators
6.Conditional Operators
7.Bitwise Operators

Arithmetic Operators

The arithmetic operator perform arithmetic operation and can be classified into unary and binary arithmetic operator. The arithmetic operator can operate on any built-in data type.A list of arithmetic operator and their meaning are given below-

Operator Meaning
+ Addition or Unary plus
- Subtraction or Unary minus
* Multiplication
/ Division
% Modulo division(reminder after division)

Integer arithmetic-When two operands such as a and b are declared integers,an arithmetic operation performed on these integer is called integer arithmetic.It always yields an integer value.For example,if a and b are integers,then for a=14 and b=4,the results are-

		a-b=10
		a+b=18
		a*b=56
		a/b=3
		a%b=2(remainder of division)

During integer division,if both the operands are of the same sign,the results is truncated towards zero.If one of them is negative the direction of truncation is implementation dependent.That is, 6/8=0 and -6/-8=0 but -6/8 may be zero or -1

Floating Point/Real Arithmetic-An arithmetic operation involving only real operands is called real arithmetic.A real operand may assume value either in decimal or exponential notation.The floating point results are rounded off to the number of significant digit specified and hence the final value is only an approximation of the correct results.The operator % cannot be used with real operands. For example,if p,q and r are floats,then-

		p=6.0/7.0=.857143
		q=1.0/3.0=.333333
		r=-2.0/3.0=.666667

Mixed Mode Arithmetic-In mixed mode arithmetic,if either of the operand is real,the resultant is always a real value.Foe example,

		15/10=1.5
Relational Operators

Relational operators are used to compares arithmetic,logical and character expression.We often compares two similar quantities and depending on their relation,take some decisions.These comparison can be done with the help of relational operators. Each of these operators compares their left hand side with their right hand side.The whole expression involving the relational operator then evaluates to an integers.It evalute to zero if the condition is false and 1 if it is true.

Operator Meaning
< Less than
> Greater than
<= Less than or equal to
>= Greater than or equal to
== Equal to
!= Not equal to
Logical Operators

A logical operator is used to compare or evaluate logical or relational expression.There are three logical operators.They are-

Operator Meaning
&& Logical AND
|| Logical OR
! Logical NOT

An expression involving && or || sometimes called logical expression or compound expression.A logical expression also yields a value of one or zero.

Logical AND-Consider the following example

	a>b&&x==10

This expression evaluates to true(1) only if both expression,a>b and x==10,are true.

Logical OR-Consider the following example,

	a<n||a<n

This expression is true if one of them is true,or if both of them are true

Logical NOT-The !(NOT) operator takes a single expression and evaluates to true(1) if the expression is false(zero) and evaluates to false(0) if the expression is true.In other words,it just reverses the value of the expression.For example

	!(x>=y)

In this expression evaluates to true only if the value of x is neither greater main nor equal to y.Therefore the whole expression including the ! operator is the same as-x

Assignment Operators

Assignment operator(=)are used to assign the result of an expression to a variable.Python has a set of shorthand assignment operator of the form,v op=exp; where v is a variable exp is an expression and op is a Python binary arithmetic operator.The operator op- is knoen as a shorthand assignment operator.The assignment statement
vop=exp; is equivalent to
v=v op(exp);
with v evaluated only once. For example
x+=y+1; This is same as the statement
x=x+(y+1); Some of the commonly used shorthand operators are-

statement with the simple operator statement with shorthand assignment operator
a=a+1; a+=1;
a=a-1; a-=1;
a=a*(n+1); a*=n+1;
a=a/(n+1) a/=n+1;
a=a%b; a%=b;
Increment or Decrement Operators

The increment and decrement operator are very useful in Python language.The syntex of the operator is given below-

		++<variable name>
		--<variable name>
		<variable name>++
		<variable name>--

The operator ++ adds 1 to the operand and -- subtracts 1 from the append.Theseoperator come in two forms-prefix and postfix.For example,++ operator can be used in two ways: ++m and m++.

Conditional Operator(Ternary Operator)

The conditional operator consists of two symbols-the question mark(?) and the colon(:).The syntex of ternary operator is
exp1? exp2: exp3; where exp1,exp2 and exp3 are expressions.

The operator?: works as follow-exp1 is evaluated first.If it is nonzero(true0,then the expression 2 is evaluated and becomes the value of the expression.Here, only one of the expression (either exp2 or exp3) is evaluated.For example

		a=10;
		b=15;
		x=(a>b)?a:b    Here,x will be assigned the value of b.
Bitwise Operator

A bitwise operator operates on each bit of data.These operators are used for testing,complementing or shifting bits to the right or left.Bitwise operators may not be applied to float or double.A list of bitwise operators are given below-

Operator Meaning
& bitwise AND
| bitwise OR
^ bitwise XOR
<< shift left
>> shift right
~ bitwise complement
