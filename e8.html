Python Program to Find Factorial of a Number

The Factorial of a positive integer n is equal to 1*2*3...n.You will learn to calculate factorial of a number using For Loop in this example.

The Factorial of negative number doesn't exist.
And,the factorial of 0 is 1, 0!=1

Example.Find Factorial of a Number

Source Code
	# Python program to find the factorial of a number provided by the user.
	num = 7
	#num = int(input("Enter a number: "))

	factorial = 1
	if num < 0:
		print("Sorry, factorial does not exist for negative numbers")
	elif num == 0:
		print("The factorial of 0 is 1")
	else:
		for i in range(1,num + 1):
			factorial = factorial*i
		print("The factorial of",num,"is",factorial)

Output

	enter a value 5
		factorial of 5=120

This program takes a positive integer from the user and computer factorial using for loop.

Since the factorial of a number may be very large the type of factorial variable is declared as unsigned long int.

If the user enter negative number, the program display error message.
