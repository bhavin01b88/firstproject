Python program to reverse a Number

Example to reverse an integer entered by the user.This problem is solved using while loop in this example.

Example.Python program to reverse a number using While loop

Source Code
	# Python Program to Reverse a Number using While loop    
     
	Number = int(input("Please Enter any Number: "))    
	Reverse = 0    
	while(Number > 0):    
		Reminder = Number %10    
		Reverse = (Reverse *10) + Reminder    
		Number = Number //10    
     
		print("\n Reverse of entered number is = %d" %Reverse)

Output

	Enter a value   789
	Reverse number= 987

This program takes an integer input from the user.Then the While loop is used until n!=0 is false.

In each iteration of while loop, the remainder when n is divided by 10 is calculated and the value of n is reduced by times.
